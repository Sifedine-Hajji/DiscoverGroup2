C'est quoi le border-image

cette propriété css permet d'appliquer une image sur une bordure

c'est la version raccourcie de border-image-source, border-image-width, border-image-repeat, border-image-slice, border-image-outset

  • border-image-source : L'image source.
  • border-image-slice : La façon dont l'image est découpée en zones (jusqu'à 4).
  • border-image-outset : La distance entre la bordure et le bord de l'élément.
  • border-image-repeat : La façon dont l'image source est adaptée afin de correspondre aux dimensions de la bordure.
  • border-image-width : La largeur de la bordure avec l'image.
